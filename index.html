ShroomieStudio

Indie UI & Game Assets for Unity Developers

About

Hello! I’m Kristina Mironenko, the creator of ShroomieStudio — a cozy little studio focused on designing intuitive UI kits and fun game templates for Unity developers. All assets are handcrafted with care and passion.

Assets

Tic Tac Toe Game Template

A full-featured 2D Tic Tac Toe game template with animated symbols, two game modes (PvP & AI), smooth UI and sound effects.

  • AI with Minimax
  • Touch & desktop ready
  • Clean C# scripts

Wooden UI Pack

A cozy UI set for mobile & desktop games. Includes checkboxes, toggles, dropdowns, sliders, progress bars, popups and more.

Popup Manager

Simple prefab system for reusable popup panels. Includes info windows, settings panels, input fields and custom animated buttons.
